Özellikler:

MERN Stack: MongoDB veritabanı, Express.js sunucu tarafı framework'ü, React kullanıcı arayüzü ve Node.js sunucu tarafı çalışma zamanı kullanılarak tam bir MERN yığını uygulamasıdır.

CRUD İşlemleri: Temel CRUD (Create, Read, Update, Delete) işlemleri, kullanıcıların verileri yönetmelerine olanak tanır.

JWT Kimlik Doğrulama: JSON Web Token kullanılarak güvenli bir kimlik doğrulama sistemi entegre edilmiştir. Kullanıcılar, güvenli bir şekilde uygulamaya kaydolabilir, giriş yapabilir ve oturumlarını yönetebilirler.

Responsive Kullanıcı Arayüzü: React kullanılarak oluşturulan kullanıcı arayüzü, farklı cihazlarda sorunsuz çalışacak şekilde tasarlanmıştır.
